Expert Bureau WordPress Theme
================================

This is a WordPress theme that replicates the original HTML-based website "Первое Экспертное Бюро".

Features:
- Complete replication of the original HTML site design and functionality
- Responsive layout matching the original site
- Custom post types for services, documents, and contacts
- Customizable contact information through WordPress Customizer
- Proper integration with WordPress navigation system

Installation Instructions:
1. Upload this theme folder to your WordPress installation in the /wp-content/themes/ directory
2. Activate the theme in your WordPress admin panel under Appearance > Themes
3. Go to Customize > Contact Information to update your contact details
4. Create pages for Services, Documents, and Contacts with appropriate slugs (uslugi, dokumenty, kontakty)
5. Set up navigation menus in Appearance > Menus

Page Setup:
- Home page: Uses index.php template
- Services page: Should use page-services.php template with slug "uslugi"
- Documents page: Should use page-documents.php template with slug "dokumenty"
- Contacts page: Should use page-contacts.php template with slug "kontakty"

The theme includes all original images, fonts, and scripts from the HTML version.
